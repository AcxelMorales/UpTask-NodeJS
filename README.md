# Uptask

## Requerimientos
- NodeJS
- Pug

## Ver versiones
``
node --version
``

## Ejecutar proyectos
### Servidor de node
``
npm start
``

### Servidor de node desarrollo
``
npm run dev
``

### Servidor de node testing
``
npm run test
``
